    John Deere 331G Compact Track Loader Features

    Engine

    Final Tier 4 (FT4)/EU Stage IV diesel engine has plenty of torque and power.

    The innovative Tri-Cool(TM) design is resistant to debris buildup and uses large coolers to provide superior performance under extreme load conditions. Include a reversible fan to ensure rapid core cleaning for high-debris situations.

    Vertical Lift Boom

    Vertical-lift booms provide greater lift height and extend to the highest point in the path for lifting to provide more stability and better truck-loading capability.

    EH Boom Performance Package

    Performance packages that are installed in the factory offer increased productivity and user-friendliness using the following features:

    • Return to Dig permits the user to place the bucket or any other attachments in their designated ready-to-work position.
    • Return To Carry allows the user to use joystick detents that automatically return the boom to a specified ready-to-carry point.
    • Boom Height Kickout lets the user choose a pre-determined boom height based on the height of the truck or hopper. It also allows you to utilize joystick detents to stop automatically at a certain height.
    • Electronic Self Level lets the self-level function on the boom upward and down, unlike the hydraulic self-level feature, which only applies to booms up.

    Stability

    Boom design allows for excellent stability by keeping the load closer to the machine. A low center of gravity, 35-degree bucket rollback, optional ride control, and Self-Leveling help keep buckets full.

    Uptime

    You can get more done between fill-ups with a large tank and fuel-efficient diesel.

    Serviceability

    Tilt the hood to access dipsticks, fuel- or fluid-filler necks, and filters at ground level.

    Onboard Grade Indication Option

    This option is available in-factory and in the field. It displays the machine’s cross-slope, roll, rainfall slope, or pitch in the in-cab monitor display.

    • Easy-to-use
    • Interface in real-time
    • Roll and pitch can be viewed as “absolute” or “relative” values
    • Viewable in degrees or percent, roll, and pitch

    Cab & Controls

    A large entryway with a tightly sealed swing-out door and ample foot and leg room make it easy to get in and comfortable.

    For greater operating flexibility, industry-exclusive three-way switchable controls with electrohydraulic joystick performance package.

    Attachment-Ready

    The universal Quik-Watch(TM) allows you to go from compact loader bucket to forks to any other item in seconds. Every loader model has enough standard hydraulic power to power drills and hydraulic hammers as well as roller levels, trenchers, power rakes, and hydraulic hammers.
